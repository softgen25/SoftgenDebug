<h1 align="center">Softgen</h1>

###

<p align="left">Softgen es un sistema de gestión de informes para equipos de aire acondicionado.<br>Es un sistema el cual abarca desde la gestión de usuarios y roles atravez de un dahsboard hasta la creación del informe en formato pdf.</p>

###

<h2 align="left">Funciones del sistema</h2>

###

<p align="left">✨ Gestion de usuarios, roles y permisos.<br>📚 Gestión de equipos.<br>📄Creación y Gestión de informes en formato PDF.</p>
